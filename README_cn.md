# sockethttp
 一个轻量级的HTTP/1.1客户端

### 环境要求:

 Python2.7及以上

### 安装sockethttp:

 在`命令提示符（管理员）`中输入 `python -m pip install sockethttp` 即可。如果需要升级已有版本，请输入 `python -m pip install --upgrade sockethttp`

### 已支持的HTTP请求类型: 

 - GET
 - POST (__仍在开发阶段，并不稳定__)

### 功能

 - 轻量级的HTTP请求系统
 - 易用且开发成本低的扩展系统
